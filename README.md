# 🚀 Productivity Manager

A modern productivity web app to manage tasks, goals, and focus sessions with a built-in Pomodoro timer.

## ✨ Features

- 📅 Interactive calendar
- ✅ Task management with priorities
- 🔍 Search & filters
- 🧠 Pomodoro timer (25/5 cycles)
- 🎯 Daily goals tracking
- 📝 Notes with Markdown preview
- 💾 Local storage (data saved automatically)
- 🌙 Dark / Light theme
